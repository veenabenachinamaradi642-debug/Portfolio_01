## ⚙️ Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/veenabenachinamaradi642-debug/Portfolio_01.git
```

Navigate to the project directory:

```bash
cd Portfolio_01
```

Run a local server:

```bash
python -m http.server 8000
```

Open your browser and visit:

```
http://localhost:8000
```
