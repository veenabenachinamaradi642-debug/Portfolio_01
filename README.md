# Professional Portfolio - Veena R Benachinamaradi

This repository contains the source code for the professional portfolio website of **Veena R Benachinamaradi**, an Electronics & Communication Engineering graduate, Python Developer, and Data Scientist.

## Live Site
The website is hosted live on GitHub Pages:
[https://veenabenachinamaradi642-debug.github.io/Portfolio-/](https://veenabenachinamaradi642-debug.github.io/Portfolio-/)

## Tech Stack
- **Structure:** HTML5 (Semantic markup)
- **Styling:** CSS3 (Modern responsive layouts, dark/light theme, glassmorphism)
- **Interactions:** Vanilla JavaScript (ES6+, Lucide Icons, interactive radar charts, custom animations)

## Features
- **Responsive Layout:** Perfect viewing on mobile, tablet, and desktop screens.
- **Dark/Light Mode:** Seamless toggle with local storage persistence.
- **Skills Radar Chart:** Interactive visualization of core strengths and competencies.
- **Project Catalog:** Highlighted work with details modals.
- **Interactive Experience Timeline:** Dynamic chronological roadmap of education and internships.
- **Contact Integration:** Functional messaging form.

## Local Development
To run this project locally:
1. Clone the repository:
   ```bash
   git clone git@github.com:veenabenachinamaradi642-debug/Portfolio-.git
   ```
2. Open `index.html` directly in a browser, or run a local server:
   ```bash
   python -m http.server 8000
   ```
3. Open `http://localhost:8000` in your web browser.
